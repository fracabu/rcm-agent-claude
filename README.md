<a name="english"></a>
## :gb: English

### Overview

A collection of Claude AI agents designed to automate property listing creation. From initial content generation to optimized, publication-ready descriptions for Booking.com and Airbnb.

### Agents

| Agent | Purpose |
|-------|---------|
| `content-creator-agent` | Generate engaging property descriptions |
| `optimizing-editor-agent` | Polish and platform-optimize listings |
| `prose-editor` | Professional editing for clarity and correctness |

### Features

- **Automated Content Generation** - Persuasive descriptions and USPs
- **Multi-Stage Refinement** - Grammar, tone, legal compliance
- **Platform-Optimized Output** - Tailored for booking platforms
- **Professional Prose** - Clarity while preserving voice

### Quick Start

```bash
# Use with Claude Code CLI
content-creator-agent: create listing for [property details]
optimizing-editor-agent: optimize for Booking.com
prose-editor: polish final draft
```
